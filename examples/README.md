# Examples and Sample Data

This directory contains sample data, mock templates, and example scripts for testing and demonstration purposes.

## Directory Structure

```
examples/
├── README.md                    # This file
├── send_sample_email.py         # Generate sample email with mock data
├── sample_data/                 # Sample artifacts and data files
│   └── sample_artifact.json     # Example daily artifact JSON
└── mock_templates/              # Mock data templates (future use)
```

## Sample Email Generator

### Usage

Generate a sample email digest with realistic mock data:

```bash
# Standard daily email format
python3 examples/send_sample_email.py

# Weekly email with visual charts (Mon/Wed/Fri style)
python3 examples/send_sample_email.py --include-visuals
```

### Output

- Creates: `/tmp/sample_email.html`
- Open in browser to preview the email format
- Shows portfolio metrics, trades, positions, and strategy performance

### Mock Data Included

- Portfolio Value: $102,450.75
- Daily P&L: +$287.45
- 4 trades (3 buys, 1 sell)
- 4 open positions
- Strategy performance breakdown
